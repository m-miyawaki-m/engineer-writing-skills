## 説明

JSON形式の構造化ログで検索性を確保し、機密情報は絶対にログに出力しない

## 具体例

```javascript
// 良い例: 構造化 + 機密情報なし
logger.info('User login', {
  userId: user.id,
  method: 'oauth',
  requestId: req.id
});
// 悪い例: 非構造化 + 機密情報あり
console.log('Login:', email, password);
```

## ポイント

パスワード・トークン・個人情報は絶対NG。`req.body` の丸ごと出力も危険
